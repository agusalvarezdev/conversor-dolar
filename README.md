# Conversor de Dólar Blue

Conversor de pesos argentinos a dólares blue y viceversa, usando datos en tiempo real.

## ¿Qué hace?
- Convierte pesos a dólares al precio blue actual
- Convierte dólares a pesos al precio blue actual
- Acepta montos con puntos y comas (ej: 1.000.000)

## Tecnologías
- Python
- requests
- API: dolarapi.com

## Cómo usarlo
1. Instalar requests: pip install requests
2. Correr: python conversor.py
3. Elegir la conversión que querés hacer
